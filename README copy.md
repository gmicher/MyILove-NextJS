# MyILove (Next.js)

Port do seu site para Next.js (App Router).

## Como rodar
```bash
npm install
npm run dev
```

## Estrutura
- `app/` rotas e páginas
- `components/Sidebar.tsx` navegação lateral
- Estado persistido no `localStorage` com hooks client-only

> Dica: Para usar os ícones, já incluímos `lucide-react`.
